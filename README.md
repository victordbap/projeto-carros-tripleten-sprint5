# Dashboard de Análise de Veículos - Sprint 5

Este projeto é um aplicativo web interativo construído com **Streamlit** para analisar dados de anúncios de vendas de carros.

## Funcionalidades
- Visualização de histogramas de quilometragem (odômetro).
- Gráficos de dispersão correlacionando preço e quilometragem.
- Botões interativos para alternar entre diferentes visualizações de dados.

## Ferramentas utilizadas
- **Linguagem:** Python
- **Bibliotecas:** Pandas, Plotly Express, Streamlit
- **Deploy:** Render

## Como rodar o projeto localmente
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o comando: `streamlit run app.py`
